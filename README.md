# Pathfinders Submission for Pathway Hackathon 2025

## Running locally:

### Clone the repository
```
git clone git@github.com:adithya-ananth/Pathway-Hackathon-2025.git
```
### Run the backend
```
cd backend
pip install -r requirements.txt
uvicorn main:app --port 8080
```
### Run the frontend
```
cd frontend
npm install
npm run dev
```
